Expand description

Trait for providing rust with a typed pointer to an R scalar/vector type.

R data are hidden behind SEXP opaque pointers with an identifier for the underlying r storage type. Thus this trait is only meaningful for SEXP-wrappers.

Marker for the R object modes / types represented as a value.
